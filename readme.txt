Plugins to add javascript library for KBBI (Kateglo)

== Description ==

Plugins ini memberikan fasilitas definisi kata di blog Anda. Pengunjung tinggal mengklik ganda satu kata, maka akan muncul popup berisi definisi kata tersebut menurut KBBI.


== Installation ==


1. Upload folder `badudu-js` ke direktori `/wp-content/plugins/`
2. Activate  plugin melalui 'Plugins' menu di WordPress Admin

== Frequently Asked Questions ==

= Bagaimana Menggunakan Plugins Ini? =

Setelah plugins terinstall, pengunjung tinggal mengklik ganda kata yang ingin dicari definisinya di KBBI.

= Popup tidak bisa menutup, bagaimanakah? =

Apakah Anda sudah menginstall plugins versi terakhir? Jika belum, silakan install ulang plugins Anda.
